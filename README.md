# Elevator Problem

## Installation
1. Clone the repo
2. `npm install`
3. `npm start`

To test:
`npm test`

## Example scenario

### Input
Sequences:
- t1: [1,2,7,3]
- t2: []
- t3: [2,3,9,1]
- t4: [5,4,6]

### Output

#### Expected
```
Path: 1 -> 2 -> 3 -> 7 -> 9 -> 3 -> 2 -> 1 -> 4 -> 5 -> 6
# floors passed: 21
# direction changes: 2
```

#### Actual
```
Path: 1 -> 2 -> 3 -> 7 -> 9 -> 3 -> 2 -> 1 -> 4 -> 5 -> 6
# floors passed: 21
# direction changes: 2
```
